# Jumpx Slides — 项目记忆

AI 幻灯片生成 WebApp 的界面外壳(非幻灯片成品)。中文界面,受众=中文训练营学员(非设计师)。核心理念:画布为主、AI 为副驾,刻意避开"紫色渐变 SaaS 聊天框"。

## 设计系统(已定)
- 基调:纸感工作室,暖白象牙纸面,大留白,发丝分隔线,无渐变堆叠。
- 颜色:--paper-0 #FAF8F2(App底) / --paper-1 #FFFFFF(卡片·幻灯片) / --paper-2 #F1EEE4(桌面) / --hair #E4DFD0(分隔线);文字 --ink #23251F / --ink-2 #5E5F54 / --ink-3 #94927F。
- 强调:墨绿 --pine #2C5E48 / --pine-soft #E7EFE8,只在阶段、主操作、AI高亮处克制使用。极少量暖辅色 --amber #B07A2E(进行中/提示)。
- 字体:中文 Noto Sans SC(思源黑体);数字/标签 Space Grotesk;活动流时间戳 Space Mono。
- 阶段流程:输入 → 大纲 → 选模板 → 渲染 → 完成。

## 布局基底(已定 = 工作台方向B + 副驾A)
- 舞台优先:当前页做成"裱框"主角(暖桌面 + 软投影),缩略图为底部横排胶片轨(16:9 需要横向宽度)。
- 右侧副驾走克制版,优先级:① 三态待办(置顶)② 对话框(常驻底部)③ 活动流"它在干什么"(可折叠)。
- 交互:生成中自动展开活动流(高光时刻),完成后自动收起。

## 5 屏清单
1. 主工作台 ✅ (Jumpx Slides/主工作台.html — 含A/B对比,已定B基底)
2. 输入起步页 (进行中)
3. 大纲编辑器(左树右预览)
4. 选模板(缩略图网格,7套,推荐预选)
5. 选输出形态(HTML vs AI配图 二选一卡片)
